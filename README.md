## Installation

Clone Project

```bash
  git clone https://github.com/attmhd/data_analisis_python.git
```

Move to Project

```bash
  cd data_analisis_python/
``` 
Install all libraries needed

```bash
  pip install -r requirements.txt
``` 
Run Streamlit

```bash
  streamlit run /dashboard/dashboard.py
```
